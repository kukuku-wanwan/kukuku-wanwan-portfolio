自己紹介と本サイト構築の背景

まずは Kukuku の自己紹介をさせてください。
私は SIer で働いており、現在 5 年目です。（2021 年 10 月現在） 1 ~ 2 年目は、HTML, CSS (Bootstrap), JavaScript (jQuery), Azure を使って、社内向けの API ポータルサイトの構築をしました。 また、社内のサーバやネットワーク管理も行っていたので、ドメコンの運用やルータの設定変更も経験しました。 （このころにブロードキャストストームと呼ばれる、絶対にやってはいけない現象を引き起こした苦い記憶があります。。。）
3 ~ 5年目は、旅行系 Web サイトの構築に携わり、現在も継続中です。 この期間は要件定義・基本設計・総合試験をしており、プログラムを書くことはめっきりなくなってしまいました。 実際にプログラムを書くのは協力会社さんなのです。（多重下請け構造の闇とかなんとか言われているものですね。。。）

ということで、Excel や Word と向き合うことが多くなってしまい、このままでは自分の技術力の成長はないのではないかと悩みに悩み抜いた末、このサイトを作ることにしました。 自分の時間を使って Web サイトを作ったり、3D モデルを作ったり、いろいろなものを作っていきたいと思っています。 成果物は Works ページに載せていきますので、お暇があればぜひご覧になってください。 日々研鑽を積んでいるエンジニアの皆さんと交流できればいいなと思います。 よろしくお願いします。
